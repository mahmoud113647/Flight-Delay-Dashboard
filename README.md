# Flight Delay Dashboard

A Power BI dashboard project for analyzing flight delays, airport performance, and airline delay patterns.

## Project Description
This dashboard was built to explore flight delay data and present it in a simple interactive way using Power BI.

## Tools and Technologies
- Power BI Desktop
- DAX
- CSV File

## Main KPIs
- Total Flights
- Delayed Flights
- On-Time Flights
- Delay %
- Average Delay Time

## Visualizations
- Top Airlines Delay
- Delay by Airport
- Delayed Flights by Day of Month
- Delay Reason Breakdown

## Purpose
This project helps users understand:
- which airlines have more delays
- which airports are affected the most
- when delays increase during the month
- what causes most delays
